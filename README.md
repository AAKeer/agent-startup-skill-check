# agent-startup-skill-check

A lightweight AI context that turns Skill usage into a natural habit.

简体中文

## Overview

AI agents often ignore existing Skills or treat them as optional references.

This project provides a lightweight project context that encourages agents to naturally discover and leverage relevant Skills before starting any task.

Rather than introducing complex routing systems or heavy workflows, this project focuses on a single principle:

Before starting a task, first consider whether existing Skills can help.

The goal is to make Skill usage a natural part of every workflow.

## Features

Lightweight and minimal

Works across any domain

Encourages proactive Skill discovery

No complex routing logic

No workflow dependencies

Easy to integrate into existing projects

## Installation

Installation is intentionally simple.

Copy the URL of this repository.

Paste it into your AI agent workspace.

Allow the agent to install and load the context automatically.

No additional configuration is required.

## Philosophy

Skills should not be treated as optional references.

Skills should be treated as reusable knowledge.

This project exists to help AI agents naturally develop that habit.

## 项目简介

很多 AI Agent 在工作时会忽略已有的 Skills，或者仅仅把它们当作可有可无的参考资料。

本项目提供了一个轻量级的项目上下文，引导 AI 在开始任何任务之前，自然地发现并利用已有的 Skills。

它并不依赖复杂的路由系统，也不需要繁琐的工作流。

它只坚持一个简单的原则：

«在开始任何任务之前，先思考已有的 Skills 是否能够帮助完成当前工作。»

希望让 Skills 的使用成为工作流程中的自然步骤，而不是事后才想到。
